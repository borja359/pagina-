Concavidad y Puntos de Inflexión

Explora cómo la segunda derivada revela la "curvatura" de una función

¿Qué significa concavidad?

Imagina que la curva es un camino. Si el camino se dobla hacia arriba como una sonrisa, decimos que es cóncava hacia arriba. Si se dobla hacia abajo como una cara triste, es cóncava hacia abajo. La segunda derivada f''(x) es el detector de curvatura: su signo nos dice exactamente qué forma tiene la curva en cada punto. Cuando f''(x) cambia de signo, la curva cambia de "humor": ese momento exacto se llama Punto de Inflexión.

😊

Cóncava hacia Arriba

f''(x) > 0
La curva "sostiene agua".
La pendiente va aumentando.

🙁

Cóncava hacia Abajo

f''(x) < 0
La curva "derrama agua".
La pendiente va disminuyendo.

🔄

Punto de Inflexión

f''(x) = 0
La curva cambia de curvatura.
El "humor" cambia aquí.

¿Cómo se calcula?

Para encontrar puntos de inflexión: 1) Calcula f″(x). 2) Iguala f″(x) = 0 y resuelve para x. 3) Verifica que el signo de f″(x) cambie a ambos lados de ese punto — si no cambia, no es un punto de inflexión.

Para f(x) = x³ − 3x: f″(x) = 6x = 0 → x = 0. A la izquierda f″(−1) = −6 < 0 (cóncava abajo), a la derecha f″(1) = 6 > 0 (cóncava arriba). ✓ El signo cambia, así que x = 0 es un punto de inflexión.
